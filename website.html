THE CUSTOM EVENT WEBPAGE: RIDE 4 FREEDOM

The Client's finished event webpage needed to be easy-to-use, user-friendly, interactive, and engaging to tell the story of the work The Liberty Project does in Mozambique, as well as encourage donations and to sign up and ride in the fundraiser.

Bringing the Web Solution to Life

The event webpage showcases a simplistic design that follows The Liberty Projectâ€™s branding and predetermined styling choices, with plenty of buttons for users to be able to donate and register to the event at any section of the webpage. The event webpage makes use of third-party applications Donorbox, Eventbrite, and My Virtual Mission. These third-party applications will take the user off the main event page, to access the specific event-related content on those applications. The event webpage is divided into five main sections:

Introduction | Donation | The Virtual Ride | Registration | Instagram

The Introduction section focuses on introducing the fundraiser event, what the user can find on the webpage, and The Liberty Project. The Donation section focuses on collecting donations with an embedded Donorbox widget. The Virtual Ride section hosts the scrollytelling GIS Web Map and link to My Virtual Mission to view live where riders are along the even route. The Registration section is simple and encourages the user to get involved and sign up for the event through the use of a register button that brings the user to Eventbrite Finally, the Instagram section integrates The Liberty Project's Instagram account on the webpage with the last four photos posted on their profile. User's will be brought to their Instagram profile when they click on any of the photos.
