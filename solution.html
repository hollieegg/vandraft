THE CUSTOM DATABASE MANAGEMENT SYSTEM

The Client’s finished database needed to be both functional and easy to use. The team built a customized User Interface inside the database driven by custom code developed with Microsoft's Visual Basic for Applications. All functions of the database work at the touch of a button. The custom User Interface the workhorse of the database.

THE USER INTERFACE

The User Interface is divided into 3 sections based on task: importing and processing data, viewing primary tables and generating summary reports. The team also included a database backup tool and the ability to delete the source data from the database.

FUNCTIONALITY

Data Loading is simple. Click the load button, and let the interface do all the work. Once clicked a browse window opens, and the Client can load the data from any location. The team noticed right away the Eventbrite data was missing some key information, so to make sure the data functions in the database properly, a prompt feature was included in the load, letting the Client add in the information before processing the data. Once the data loads are complete, populating the primary tables with the data can be done rapidly with the Generate Data buttons.

ADDING VALUE

To maximize functionality of the database, five built-in summary reports were included. The summary reports provide the Client with quick access to key data isolated based on their business requirements. The Client will be able to easily create PDF print outs of event shirt sizes, or a donation summary, and exportable excel tables of event participants, mailing lists and a donor list, of course, at the touch of a button.

THE CUSTOM EVENT WEBPAGE: RIDE 4 FREEDOM

The Client's finished event webpage needed to be easy-to-use, user-friendly, interactive, and engaging to tell the story of the work The Liberty Project does in Mozambique, as well as encourage donations and to sign up and ride in the fundraiser.

Bringing the Web Solution to Life

The event webpage showcases a simplistic design that follows The Liberty Project’s branding and predetermined styling choices, with plenty of buttons for users to be able to donate and register to the event at any section of the webpage. The event webpage makes use of third-party applications Donorbox, Eventbrite, and My Virtual Mission. These third-party applications will take the user off the main event page, to access the specific event-related content on those applications. The event webpage is divided into five main sections:

Introduction | Donation | The Virtual Ride | Registration | Instagram

The Introduction section focuses on introducing the fundraiser event, what the user can find on the webpage, and The Liberty Project. The Donation section focuses on collecting donations with an embedded Donorbox widget. The Virtual Ride section hosts the scrollytelling GIS Web Map and link to My Virtual Mission to view live where riders are along the even route. The Registration section is simple and encourages the user to get involved and sign up for the event through the use of a register button that brings the user to Eventbrite Finally, the Instagram section integrates The Liberty Project's Instagram account on the webpage with the last four photos posted on their profile. User's will be brought to their Instagram profile when they click on any of the photos.

Ride 4 Freedom Webpage

Telling The Story of The Ride 4 Freedom

Using a combination of HTML, JavaScript, and CSS, Vanritto successfully created an interactive scrolling GIS Web Map which tells the story of The Liberty Project Fundraiser, Ride 4 Freedom. The series of chapters applied to the Web Map results in smooth transitioning between the stops. This allows the user to scroll effectively from one stop to another while providing information about the fundraiser, Mozambique, and The Liberty Project in the form of images and text.

The team was able to implement full customization for the route, icons and labels using Layers within Mapbox which allowed for the placing of icons upon the route line. Creating a dataset and layer in Mapbox Studio for custom labels allowed the labels to be positioned beside the icons and allowed for filtering of existing labels.

The scrollytelling Web Map is the centrepiece of the Ride 4 Freedom event webpage and celebrates the ride that supports the women of Mozambique.
